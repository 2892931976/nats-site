NATS, is an open source, modern, secure, messaging system for distributed systems and scalable cloud applications, utilized for cloud native applications, IoT device messaging, and microservices architectures.

The Apcera team maintains The NATS server which is written in Go, Python, Ruby, Node.js, Elixir, Java, NGINX, C, and C# clients. The community has contributed a growing list of libraries, including Rust, Lua, PHP and many more.

Companies like Vaidu, Siemens, VMware, HTC and Ericsson rely on the NATS enterprise messaging system for its highly performant and resilient capabilites.

NATS is open-source software, as is this site.
