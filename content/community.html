Social

Below are some example of projects and products in the ecosystem using NATS - if you would like to submit something to this list, please fork this site, and submit a pull request, or contact info@nats.io

Project or Company Description Link
Apcera Apcera's Platform uses NATS as the secure control plane www.apcera.com
AWS NATS This repository contains tools and templates to help using NATS in the Amazon Web Services platform. https://github.com/dankomiocevic/aws-nats
Bridgevine Bridgvine provides a customer acquisition technology, and NATS acts as a transaction engine in their platform http://bridgevine.com/
Carrenza Carrenza use NATS in their Paas Platform https://github.com/carrenza
CloudFoundry CloudFoundry use NATS as a control plane https://docs.cloudfoundry.org/concepts/architecture/messaging-nats.html
CodeFresh CodeFresh is an automated Docker workflow tool for Development teams https://codefresh.io/
Dokku-NATS NATS plugin for Dokku, a PaaS Platform https://github.com/dokku/dokku-nats
Fluentd Event Collector NATS plugin for the Fluentd Event Collector https://github.com/cloudfoundry-community/fluent-plugin-nats
Go-Micro A pluggable RPC microservice library in Go https://github.com/micro/go-micro
InfluxData The NATS consumer plugin reads from specified NATS subjects and adds messages to InfluxDB. The plugin expects messages in the Telegraf Input Data Formats. A Queue Group is used when subscribing to subjects so multiple instances of telegraf can read from a NATS cluster in parallel. https://github.com/influxdata/telegraf/tree/8d0f50a6fd10cea1af84ccfdc2beeb1a61dd19bd/plugins/inputs/nats_consumer
NATS Clustering NATS clustering with CoreOS, etcd2 and Docker https://github.com/pires/nats-coreos-cluster
NATS to Logstash This library will resend configured nats messages directly to logstash through a tcp connection. https://github.com/carrenza/nats_to_logstash
NATS-Mon A realtime web based NATS monitor https://github.com/repejota/nats-mon
NATSBoard Dashboard for monitoring NATS. It provides real-time information from NATS server. https://github.com/cmfatih/natsboard
Nvisibile Inc Go ARI Proxy An implementation of the [go-ari-library] used to connect to the Asterisk REST Interface for delivery of Events and Commands across a message bus. https://github.com/nvisibleinc/go-ari-proxy
Piper Push Cache Piper Push Cache is a JSON caching server that supports both WebSockets and SSE on the outgoing side and among others, NATS messages on the incoming side. http://www.piperpushcache.com/opslog_build.html
REST to NATS Proxy This library provides translation of HTTP requests to NATS messaging platform. https://github.com/sohlich/nats-proxy
Rancher NATS This is an extended version of the official Docker image for nats. It adds support to set up a cluster easily onrancher. http://bit.ly/1X36qKZ
Symfony NATS A Nats bundle for Symfony2 https://packagist.org/packages/octante/nats-bundle
Taurus Experimental Mesos framework for Docker containers written in Go https://github.com/milosgajdos83/taurus
Tlks.io A community driven curated list of tech talks http://tlks.io/
