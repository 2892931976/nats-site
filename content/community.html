Community

If you’d like to help make NATS and its resources better, here are some ways. Chances are, if you find a feature useful, others will too. This website pulls documentation from GitHub so once you update the readme on GitHub, changes will show up here as well.

People Using NATS

Below are some example of projects and products in the ecosystem using NATS - if you would like to submit something to this list, please fork this site, and submit a pull request, or contact info@nats.io

Project or Company Description
Apcera Apcera's Platform uses NATS as the secure control plane
Arduino NATS An Arduino / Spark Core / Particle Photon compatible C++ library for communicating with a NATS server.
AWS NATS This repository contains tools and templates to help using NATS in the Amazon Web Services platform.
Bridgevine Bridgvine provides a customer acquisition technology, and NATS acts as a transaction engine in their platform
Carrenza Carrenza use NATS in their Paas Platform
CloudFoundry CloudFoundry use NATS as a control plane
CodeFresh CodeFresh is an automated Docker workflow tool for Development teams
Dokku-NATS NATS plugin for Dokku, a PaaS Platform
Fluentd Event Collector NATS plugin for the Fluentd Event Collector
FluentBit The NATS output plugin allows you to flush your records into a NATS Server end-point.
Go-Micro A pluggable RPC microservice library in Go
InfluxData The NATS consumer plugin reads from specified NATS subjects and adds messages to InfluxDB. The plugin expects messages in the Telegraf Input Data Formats. A Queue Group is used when subscribing to subjects so multiple instances of telegraf can read from a NATS cluster in parallel.
Logrus-NATS A Logrus hook for sending data to NATS.
NATS Clustering NATS clustering with CoreOS, etcd2 and Docker
NATS to Logstash This library will resend configured nats messages directly to logstash through a tcp connection.
NATS-Mon A realtime web based NATS monitor
NATSBoard Dashboard for monitoring NATS. It provides real-time information from NATS server.
Nvisible Inc Go ARI Proxy An implementation of the [go-ari-library] used to connect to the Asterisk REST Interface for delivery of Events and Commands across a message bus.
Piper Push Cache Piper Push Cache is a JSON caching server that supports both WebSockets and SSE on the outgoing side and among others, NATS messages on the incoming side.
REST to NATS Proxy This library provides translation of HTTP requests to NATS messaging platform.
Rancher NATS This is an extended version of the official Docker image for nats. It adds support to set up a cluster easily onrancher.
Symfony NATS A Nats bundle for Symfony2
Taurus Experimental Mesos framework for Docker containers written in Go
Tlks.io A community driven curated list of tech talks

Connector Description Developer Support

REST to NATS Proxy This library provides a translation of HTTP requests to NATS. Client side is partly inspired by Gin to process requests Radomír Sohlich Community
